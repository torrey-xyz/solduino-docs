# Custom Program Interactions

Reference: `sol/examples/custom_program_demo/custom_program_demo.ino`.

## What This Guide Covers

- Constructing custom `Instruction` payloads.
- Encoding compact typed data in instruction buffers.
- Combining multiple instructions into one `Transaction`.
- Using helper modules like `SystemProgram` and `TokenProgram`.
- Deriving PDAs for deterministic program account addressing.
