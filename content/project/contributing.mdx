# Contributing

Contributing guidelines live in:

- `sol/CONTRIBUTING.md`
- `sol/CODE_OF_CONDUCT.md`
- `sol/SECURITY.md`

## Docs Maintenance Rules

- Keep API docs aligned with public headers in `sol/`.
- Add or update example docs whenever a new `.ino` sketch is added.
- Update changelog and versioned docs sections for every release.
